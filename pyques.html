7.What is Control Statements?

Control statements in Python are used to control the flow of a program's execution. Here are some common control statements:

    Conditional Statements
    1. If: Used to execute a block of code if a condition is true.
    2. Elif: Used to check another condition if the initial condition is false.
    3. Else: Used to execute a block of code if all conditions are false.
Loops
    1. For: Used to iterate over a sequence (such as a list, tuple, or string) and execute a block of code for each item.
    While: Used to execute a block of code as long as a condition is true.
Jump Statements
    1. Break: Used to exit a loop prematurely.
    2. Continue: Used to skip the rest of the current iteration and move on to the next one.
    3. Pass: Used as a placeholder when a statement is required syntactically but no execution of code is necessary.

8.Oops concept in Python?

Object-Oriented Programming (OOP) in Python OOP is a programming paradigm that focuses on organizing and structuring code using objects and classes. In Python, OOP concepts include:

    Key Principles:
      1. Encapsulation: Bundling data and methods that operate on that data within a single unit (class or object).
      2. Abstraction: Hiding implementation details and showing only the necessary information to the outside world.
      3. Inheritance: Creating a new class based on an existing class, inheriting its properties and behavior.
      4. Polymorphism: Ability of an object to take on multiple forms, depending on the context.


    9.Functions in Python?

    Functions are reusable blocks of code that perform a specific task. They help organize code, reduce repetition, and make programs more modular
    Defining a Function
    In Python, you define a function using the def keyword followed by the function name and parameters in parentheses. def greet(name): print(f"Hello, {name}!")
    Calling a Function
    To use a function, you call it by its name followed by arguments in parentheses. greet("John") # Output: Hello, John!

      Function Types
    1. Built-in Functions: Python has many built-in functions, such as len(), type(), and range().
    2. User-defined Functions: You can define your own functions using the def keyword.
      Function Arguments
      1. Positional Arguments: Arguments passed to a function based on their position.
      2. Keyword Arguments: Arguments passed to a function using their keyword.
      3. Default Arguments: Arguments with default values, which can be omitted when calling the function.
      4. Variable Arguments: Functions can accept a variable number of arguments using *args or **kwargs.

    10.File Input/Output (I/O) in Python

    File I/O is an essential aspect of programming, allowing you to read and write data to files.
    Reading from a File
    You can read from a file using the open() function in read mode ('r'). with open('example.txt', 'r') as file: content = file.read() print(content)
    Writing to a File
    You can write to a file using the open() function in write mode ('w'). with open('example.txt', 'w') as file: file.write('Hello, World!')
    Modes
    1. 'r': Read mode (default). 2. 'w': Write mode, truncating the file if it exists. 3. 'a': Append mode, adding to the end of the file. 4. 'x': Create mode, failing if the file exists. 5. 'b': Binary mode. 6. 't': Text mode (default).
